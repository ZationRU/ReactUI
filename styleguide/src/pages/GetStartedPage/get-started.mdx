# Installation

To use ZnUI in your project, run one of the following commands in your terminal:

import {
    SegmentedButton,
    Layout, VStack
} from "@znui/react";
import { useState } from 'react';
import {CodeFragment} from "../../styleguide/StyleGuide/CodePlaygroundDocs"

export const PackageInstall = () => {
    const [selectedPackageManager, setPackageManager] = useState('npm');

    return <VStack spacing={15}>
        <SegmentedButton w={{
            esm: 'auto',
            emd: 400
        }} selectedIds={selectedPackageManager} onSelect={setPackageManager}>
            <SegmentedButton.Segment id='npm'>
                npm
            </SegmentedButton.Segment>
            <SegmentedButton.Segment id='yarn'>
                yarn
            </SegmentedButton.Segment>
            <SegmentedButton.Segment id='pnpm'>
                pnpm
            </SegmentedButton.Segment>
            <SegmentedButton.Segment id='bun'>
                bun
            </SegmentedButton.Segment>
        </SegmentedButton>

        <CodeFragment>
            {selectedPackageManager==='npm'&&"npm install @znui/react"}
            {selectedPackageManager==='yarn'&&"yarn add @znui/react"}
            {selectedPackageManager==='pnpm'&&"pnpm add @znui/react"}
            {selectedPackageManager==='bun'&&"bun add @znui/react"}
        </CodeFragment>
    </VStack>
}

<PackageInstall/>

After installing ZnUI, you need to set up the <code>ZnUIProvider</code> and <code>ZnUIPortal</code> at the root of your application.
<code>ZnUIPortal</code> is util for dialogs and modals and must be one.

```tsx
import * as React from 'react';

import { ZnUIProvider, useSnackbar, ZnUIPortalDestination, Button } from '@znui/react'

function App() {
  return (
    <ZnUIProvider initialScheme='system'>
        <ZnUIPortalDestination/>

        <YourApplication />
    </ZnUIProvider>
  )
}
function YourApplication() {
  const snackbar = useSnackbar()
  return <Button onClick={() => snackbar("Hello, world!")}>My Cool Button in My Application</Button>
}

<App/>
```
